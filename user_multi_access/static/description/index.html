Smart Date(Time) Input

Input value in date(time) fields smartly


This module enhances the input power of date(time) fields Check video for reference Odoo Quick Tips - Set dates with shortcuts [General]

    These are new enhancement
  • Add/subtract hours and minutes from current time using the formats below
    e.g. "+1h" or "-30m"
  • Add/subtract from current time using chain inputs
    e.g. "-1y +1m +10d" (add 1 month 10 days on last year from current date)
  • Add/subtract from selected date(time), just add comma and then input chain
    e.g. "2024/01/01,+4m +10d". Results "2024/05/10"
  • Input "now", "yesterday", "tomorrow" to set the date accordingly

Formats

Years

+2year, +2y, -2yr

Months

+2months, +2m, -2m

Days

+2days, +2d, -2d

Hours

+2hours, +2h, -2hr

Minutes

+2minutes, +2min, -2x
